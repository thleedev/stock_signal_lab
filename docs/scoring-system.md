# 종목 점수 산정 기준

> 최종 업데이트: 2026-03-25

## 개요

종목분석 점수는 **4개 카테고리**의 가중합으로 산출됩니다.

```
종합 점수 = 신호(10%) + 기술(40%) + 밸류에이션(10%) + 수급(40%)
```

각 카테고리는 0~100점으로 정규화된 후 가중합으로 종합 점수(0~100)를 만들고, 이를 기반으로 등급이 부여됩니다.

### 등급 기준

| 종합 점수 | 등급 | 라벨 |
|-----------|------|------|
| 90 이상 | **A+** | 적극매수 |
| 80~89 | **A** | 매수 |
| 65~79 | **B+** | 관심 |
| 50~64 | **B** | 보통 |
| 35~49 | **C** | 관망 |
| 35 미만 | **D** | 주의 |

### 데이터 소스

| 소스 | 갱신 주기 | 용도 |
|------|-----------|------|
| 네이버 증권 API | 신호 수신 시 즉시 + 크론(20시) | 시세, 수급, 지표, Forward, 일봉 |
| Android Collector | SMS 수신 시 실시간 | BUY/SELL 신호 |
| 클라이언트 실시간 | 30초 폴링 | 장중 등락률 보정 |

---

## 1. 신호 점수 (가중치 10%)

BUY 신호의 반복성과 매수가 대비 현재 위치를 평가합니다.

### 기본 경로 (calcScore)

원점수 0~100

| 항목 | 조건 | 점수 |
|------|------|------|
| **30일 신호 횟수** | 5회 이상 | +60 |
| | 3회 이상 | +50 |
| | 2회 이상 | +40 |
| | 1회 이상 | +30 |
| **매수가 대비 갭** (보조) | 매수가 이하 (저가 진입 기회) | +20 |
| | +5% 미만 (아직 초입) | +15 |
| | +10% 미만 | +10 |
| | +20% 미만 | +5 |
| | +20% 이상 (추격매수) | **-10** |
| 매수가 정보 없음 | 신호 1회+ | +10 |

### AI 추천 경로 (calcSignalScore)

원점수 0~30 → 정규화: `÷30 ×100`

| 항목 | 조건 | 점수 |
|------|------|------|
| 오늘 BUY 소스 수 | 3개 이상 동시 | +15 |
| | 2개 | +10 |
| | 1개 | +5 |
| 오늘 신호 존재 | 보너스 | +5 |
| 30일 3회+ 반복 | 빈번 신호 | +5 |
| 현재가 ≤ 신호가 | 저가 진입 | +5 |

## 2. 기술/모멘텀 점수 (가중치 40%)

가격 위치, 차트 패턴, 섹터 상대강도를 종합 평가합니다.

### 기본 경로 (calcScore)

원점수 0~100

#### 52주 범위 위치

`position = (현재가 - 52주저점) ÷ (52주고점 - 52주저점)`

| position | 의미 | 점수 |
|----------|------|------|
| ≤ 15% | 바닥권 | +40 |
| ≤ 30% | 저점 이탈 초기 | +35 |
| ≤ 50% | 중간 하단 | +25 |
| ≤ 70% | 중간 상단 | +15 |
| ≤ 85% | 고점 접근 | +8 |
| > 85% | 52주 고점 근접 | +3 |

#### 등락률

| 등락률 | 의미 | 점수 |
|--------|------|------|
| +3~5% | 상승 초입 (최적) | +40 |
| +1~3% | 완만 상승 | +30 |
| +5~10% | 상승 진행 | +25 |
| 0~1% | 보합 | +15 |
| +10~15% | 강한 상승 | +10 |
| -3~0% | 소폭 하락 (눌림목) | +5 |
| -5~-3% | 조정 | +3 |
| -10~-5% | 급락 | 0 |
| +15~25% | 과열 주의 | **-5** |
| +25% 이상 | 극단적 급등 | **-20** |
| -10% 이하 | 폭락 | **-10** |

#### 섹터 상대강도

`상대강도 = 개별 등락률 - 같은 섹터 평균 등락률`

| 상대강도 | 의미 | 점수 |
|----------|------|------|
| +5% 이상 | 섹터 주도주 | +20 |
| +2% 이상 | 상대 강세 | +12 |
| 0% 이상 | 평균 이상 | +5 |
| -5% 이하 | 크게 뒤처짐 | **-10** |

### AI 추천 경로 (calcTechnicalScore)

원점수 -12~48 → 정규화: `(score+12) ÷60 ×100`

최소 20일 일봉 데이터 필요. 부족 시 0점 반환.

#### 가산 지표 (12개)

| 지표 | 조건 | 점수 |
|------|------|------|
| RSI 반등 구간 | RSI 30~50 | +5 |
| 골든크로스 | 5일선 > 20일선 상향돌파 (3일 내) | +5 |
| 불새패턴 (V자 반등) | 3일 음봉 후 +3% 장대양봉 | +5 |
| 이격도 반등 | 현재가/20일선 92~98% + 양봉 | +5 |
| 거래량 바닥 탈출 | 10일 평균 ≤ 20일 평균×50% → 오늘 2배+ | +5 |
| MACD 골든크로스 | 12/26/9 시그널 돌파 (3일 내) | +4 |
| 거래량 급증 | 20일 평균 대비 2배+ | +4 |
| 이평선 정배열 | 5일 > 20일 > 60일 | +4 |
| 볼린저 하단 반등 | 하단 이탈 후 복귀 (5일 내) | +4 |
| 연속하락 후 반등 | 3일+ 하락 후 +1.5% 양봉 | +4 |
| 52주 저점 근접 | 현재가/52주저점 ±5% | +3 |
| 초기진입 가산 | 골든/MACD/거래량 + 등락 0~3% | +3 |

#### 감산 지표 (3개)

| 지표 | 조건 | 점수 |
|------|------|------|
| 쌍봉 패턴 | 20일 내 이중 고점 + 현재가 2nd 고점 근접 | **-8** |
| RSI 과매수 + 급등 | RSI ≥ 70 + 5일 등락 ≥ +10% | **-6** |
| 극단적 급등 | 5일 등락 ≥ +25% | **-4** |

### 실시간 보정 (클라이언트)

장중 실시간 가격이 반영되면 모멘텀 점수를 보정합니다.

| 실시간 등락률 | AI 종목 보정 | 비AI 종목 |
|---------------|-------------|-----------|
| +5% 이상 | +15 | 전체 재계산 |
| +3% 이상 | +10 | 〃 |
| +1% 이상 | +5 | 〃 |
| -3% 이하 | -5 | 〃 |
| -5% 이하 | -10 | 〃 |

## 3. 밸류에이션 점수 (가중치 10%)

펀더멘탈 저평가 매력도를 평가합니다. Forward 컨센서스 데이터 유무에 따라 분기합니다.

### 기본 경로 (calcScore)

원점수 0~100

#### Forward 데이터 있는 경우

| 지표 | 조건 | 점수 | 최대 |
|------|------|------|------|
| **Forward PER** | <5 / <8 / <12 / <15 / <20 | 35 / 28 / 18 / 8 / 3 | 35 |
| **목표주가 상승여력** | ≥50% / ≥30% / ≥15% / ≥5% | 25 / 20 / 12 / 5 | 25 |
| **투자의견** | ≥4.5 / ≥3.5 / ≥2.5 | 15 / 10 / 3 | 15 |

#### Forward 데이터 없는 경우

| 지표 | 조건 | 점수 | 최대 |
|------|------|------|------|
| **Trailing PER** | <5 / <8 / <12 / <15 / <20 | 35 / 28 / 18 / 8 / 3 | 35 |
| **PBR** | <0.3 / <0.5 / <0.8 / <1.0 / <1.5 | 35 / 30 / 20 / 10 / 3 | 35 |

#### 공통 (항상 적용)

| 지표 | 조건 | 점수 | 최대 |
|------|------|------|------|
| **ROE** | >25% / >20% / >15% / >10% / >5% | 30 / 25 / 20 / 12 / 5 | 30 |
| **배당수익률** | ≥5% / ≥3% / ≥1.5% | 15 / 10 / 5 | 15 |

### AI 추천 경로 (calcValuationScore)

원점수 0~25 → 정규화: `÷25 ×100`

동일 구조이나 배점 축소: Forward PER 최대 8, 목표주가 최대 8, 투자의견 최대 4, PBR 최대 7, ROE 최대 6, 배당 최대 5

## 4. 수급 점수 (가중치 40%)

외국인·기관 매매 동향을 평가합니다.

### 기본 경로 (calcScore)

원점수 0~100

| 항목 | 조건 | 점수 |
|------|------|------|
| 외국인 당일 순매수 | > 0 | +20 |
| 기관 당일 순매수 | > 0 | +20 |
| 외국인 5일 누적 순매수 | > 0 | +12 |
| 기관 5일 누적 순매수 | > 0 | +12 |
| 외국인 연속매수 | 5일+ / 3일+ / 2일+ | +20 / +15 / +8 |
| 기관 연속매수 | 5일+ / 3일+ / 2일+ | +20 / +15 / +8 |
| 동반매수 시너지 | 외국인+기관 5일 동반 순매수 | +10 |
| 시총 대비 순매수 | > 0.1% / > 0.05% | +8 / +4 |
| 공매도 비율 (당일) | < 0.5% / < 1% | +10 / +5 |

### AI 추천 경로 (calcSupplyScore)

원점수 -10~45 → 정규화: `(score+10) ÷55 ×100`

기본 경로와 동일 구조이나 배점 축소. 추가로 **매도세 감점**:

| 항목 | 조건 | 점수 |
|------|------|------|
| 외국인 3일+ 연속 매도 | streak ≤ -3 | **-5** |
| 기관 3일+ 연속 매도 | streak ≤ -3 | **-5** |

## 종목별 점수 산출 예시

### 삼성전자 (005930) — 반도체 대장주

```
현재가: 182,000원 (등락률 -3.14%)
Forward PER: 7.83  목표주가: 252,720원 (상승여력 +38.9%)  투자의견: 4.0(매수)
52주: 52,000 ~ 228,500 (위치 73.6%)
수급: 외국인 4일 연속 매도(-1,514만주 5일 누적), 기관 4일 연속 매도
신호: 30일 1회 BUY, 매수가 정보 없음
```

| 카테고리 | 점수 | 산출 근거 |
|----------|------|-----------|
| 신호 | 40 | 1회(+30) + 매수가없음(+10) |
| 기술 | 18 | 52주 위치 73%(+15) + 등락률 -3.14%(+3) + 섹터 상대강도 미미 |
| 밸류에이션 | 58 | Forward PER 7.83<8(+28) + 상승여력 38.9%≥30%(+20) + 투자의견 4.0(+10) |
| 수급 | 0 | 외국인·기관 모두 순매도, 연속매수 없음 |
| **종합** | **40×0.1 + 18×0.4 + 58×0.1 + 0×0.4 = 17.0** | **→ D** |

> 밸류에이션은 우수하나, 외국인·기관 대량 매도로 수급(40%)이 0점이어서 D. 수급 반전 시 급등 가능.

### SK하이닉스 (000660) — AI 반도체

```
현재가: 900,000원 (등락률 -3.23%)
Forward PER: 5.36  목표주가: 1,335,200원 (상승여력 +48.4%)  투자의견: 4.0(매수)
52주: 162,700 ~ 1,117,000 (위치 77.2%)
수급: 외국인 4일 연속 매도, 기관 5일 누적 순매수(+49,166)
신호: 30일 1회 BUY
```

| 카테고리 | 점수 | 산출 근거 |
|----------|------|-----------|
| 신호 | 40 | 1회(+30) + 매수가없음(+10) |
| 기술 | 18 | 52주 위치 77%(+8) + 등락률 -3.23%(+3) + 섹터 미미 |
| 밸류에이션 | 63 | Forward PER 5.36<8(+28) + 상승여력 48.4%≥30%(+20) + 투자의견 4.0(+10) + 배당 미미 |
| 수급 | 12 | 기관 5일 누적(+12), 외국인 매도로 상쇄 |
| **종합** | **40×0.1 + 18×0.4 + 63×0.1 + 12×0.4 = 22.3** | **→ D** |

> Forward PER 5.36배로 극저평가이나, 밸류에이션 가중치가 10%뿐이라 종합 점수에 미반영. 외국인 대량 매도가 수급 점수를 압도.

### 한국항공우주 (047810) — 방산

```
현재가: 179,400원 (등락률 +2.28%)
Forward PER: 47.94  목표주가: 185,111원 (상승여력 +3.2%)  투자의견: 3.78
52주: 70,600 ~ 215,500 (위치 75.1%)
수급: 외국인 5일 연속 매도, 기관 당일 순매수

```

|----------|------|-----------|
| 신호 | 45 | 1회(+30) + 매수가 이하~+5%(+15) |
| 기술 | 53 | 52주 위치 75%(+8) + 등락률 +2.28%(+30) + 섹터 상대강도(방산 강세) |

| 수급 | 20 | 기관 당일(+20), 외국인 5일 연속 매도 |

> 섹터 모멘텀(방산)으로 기술 점수 상승. 그러나 Forward PER 48배로 고평가, 목표주가 거의 도달.

### LIG넥스원 (079550) — 방산 (천궁-II)

```

Forward PER: 43.08  목표주가: 700,000원 (상승여력 -5.8%)  투자의견: 4.0
52주: 220,000 ~ 940,000 (위치 72.8%)
수급: 외국인 당일 매도, 기관 당일 순매수(+15,093), 기관 2일 연속

```

| 카테고리 | 점수 | 산출 근거 |
|----------|------|-----------|
| 신호 | 50 | 2회(+40) + 매수가 대비 +5.5%<10%(+10) |
| 기술 | 23 | 52주 위치 73%(+8) + 등락률 +0.13%(+15) + 섹터 미미(stale) |

## 점수 체계 특성 및 한계

### 강점
- 매수가 대비 갭으로 "지금 진입해도 되는가"를 정량적으로 평가
- 섹터 모멘텀으로 테마 급등(방산, AI 등) 포착
- 외국인/기관 연속매도 감점으로 스마트머니 이탈 반영
- 실시간 등락률 보정으로 장중 급등/급락 즉시 반영

### 한계
- **뉴스/이벤트 미반영**: 수주 공시, 실적 서프라이즈, 지정학적 이벤트는 점수에 반영되지 않음
- **Forward 데이터 커버리지**: 애널리스트 커버리지가 없는 중소형주는 Forward PER/목표주가가 null → 밸류에이션 폴백
- **가중치 편중**: 기술(40%)+수급(40%)=80%로, 수급이 없으면 최대 점수가 20점으로 제한됨
- **stock_cache 갱신 주기**: 크론 20시 + 신호 수신 시 enrichment. 장중 데이터는 실시간 보정으로 부분 커버
