# 종목추천 & 단기추천 — 순위산출 로직 (2026-03-27)

> 이 문서는 현재 운영 중인 점수 체계를 기록한다.
> 가중치·점수 조정 시 이 문서를 함께 업데이트할 것.

---

## 1. 공통 파이프라인

```
signals 테이블 (오늘 BUY/BUY_FORECAST)
  └─ fetchTodayBuySymbols()
       └─ 후보 종목 배열
            ├─ stock_cache (재무/수급 캐시)
            ├─ daily_prices (65일 일봉)
            ├─ naver 투자자 데이터 (캐시 미스 시 실시간)
            └─ krx forward 데이터
```

두 모델 모두 **오늘 BUY 신호가 발생한 종목만** 후보로 사용한다.

## 2. 종목추천 (Standard Model)

### 2-1. 가중치 — 시총 티어별

| 구성요소 | 대형주 | 중형주 | 소형주 | 원점수 범위 |
|----------|--------|--------|--------|------------|
| signal (신호 강도) | 5% | 8% | 10% | 0~30 |
| trend (기술적 분석) | 28% | 35% | 45% | 0~65 |
| valuation (밸류에이션) | 15% | 20% | 22% | 0~25 |
| supply (수급) | 22% | 18% | 23% | -10~45 |
| earnings_momentum (이익모멘텀) | 30% | 19% | 0% | 0~100 |
| risk (감산) | 15% | 15% | 15% | 0~100 |

> **소형주는 이익모멘텀 0%** — Forward 데이터가 부족하므로 trend·supply 중심.

### 2-2. 총점 공식

```
base =
  (signal / 30)  × w_signal +
  (trend  / 65)  × w_trend +
  (valuation / 25) × w_valuation +
  ((supply + 10) / 55) × w_supply +
  (earnings_momentum / 100) × w_earnings_momentum

total = clamp(0, base − (risk / 100) × w_risk, 100)
```

| 조건 | 가점 |
|------|------|
| 오늘 BUY 소스 3개 동시 | +15 |
| 오늘 BUY 소스 2개 | +10 |
| 오늘 BUY 소스 1개 | +5 |
| 오늘 BUY 신호 존재 | +5 |
| 최근 30일 BUY 3회 이상 | +5 |
| 현재가 ≤ 신호가 (저점 매수 가능) | +5 |

**파일:** `signal-score.ts`

| 거래량 급증 (20일 평균 2배+) | +4 | |

| 조건 | 가점 | 비고 |
|------|------|------|
| 볼린저 하단 이탈 후 복귀 | **+6** | 평균회귀 트리거 |

| 이격도 반등 (기본: 92~98% + 양봉) | **+5** | |
| 연속하락 5일+ 후 반등 +1.5% | **+8** | 하락 깊을수록 가점 |

## 3. 단기추천 (Short-Term Momentum Model)

> 목표: 1~2일 내 주가 상승 확률이 높은 종목 선별

### 3-1. 가중치

| 구성요소 | 가중치 | 원점수 범위 |
|----------|--------|------------|
| momentum (가격×거래량) | 38% | -10~90 |
| supply (수급) | 22% | -25~55 |
| catalyst (촉매) | 28% | -10~60 |
| valuation (밸류에이션) | 7% | 0~25 |
| risk (감산) | 18% | 0~100 |

### 3-2. 총점 공식

```
각 원점수 → 정규화 (0~100)
base = momentum_norm × 0.38 + supply_norm × 0.22 + catalyst_norm × 0.28 + valuation_norm × 0.07
total = clamp(0, base − risk_norm × 0.18, 100)
```

---

### 3-3. 프리필터 (1차 필터)

후보 종목 중 아래 조건을 **모두** 충족해야 스코어링 단계에 진입:

| 조건 | 기본 | 강한 촉매 시 |
|------|------|-------------|
| 등락률 | ≥ +0.5%, < +8% | **≥ -1%**, < +8% |
| 거래대금 | ≥ 200억 | ≥ 200억 (고정) |
| 종가위치 | ≥ 0.5 | **≥ 0.4** |
| 수급 | 외국인/기관 1개+ 순매수 | 동일 |
| 과열 | 3일 누적 ≤ 20% | 동일 |
| 촉매 | 3일 내 BUY or 섹터 강세 | 동일 |

> **강한 촉매** = BUY 소스 2개+ 동시 발생 또는 섹터 상승률 상위 3

**파일:** `short-term/pre-filter.ts`

---

### 3-4. 모멘텀 점수 — 원점수 -10~90

#### A. 가격×거래량 매트릭스 (최대 35)

| 등락률 \ 거래량 배수 | ≥ 2배 | [1.5, 2) | < 1.5 |
|---------------------|-------|----------|-------|
| [+1%, +3%) 최적 | **+35** | +28 | +18 |
| [+3%, +6%) | +30 | +25 | +15 |

| [-1%, +0.5%) | +12 | +8 | +3 |
| [+6%, +8%) | +15 | +12 | +5 |
| ≥ +8% (과열) | **-5** | -8 | -10 |

#### B. 종가위치 (최대 20)

| 종가위치 | 점수 |
|---------|------|
| ≥ 0.8 | +20 |
| ≥ 0.6 | +12 |

| ≥ 0.3 | 0 |

#### C. 갭업 + 패턴 (합산 후 clamp 20)

|-----------------------|------|
| 전일 장대양봉 + 소폭 갭업 | +20 |
| 전일 장대양봉 + 보합 출발 | +15 |
| 갭업 [+1%, +3%) | +12 |
| 갭업 ≥ +3% | +3 |

## 4. 설계 원칙

1. **저점매수 우선**: 기술점수에서 저점 식별 시그널(볼린저 복귀, 이격도 반등, 연속하락 반등)이 추세추종(정배열, 추세지속) 대비 동등하거나 높은 가점
2. **수급 초기진입**: streak=1(매수 전환 첫날)에 장기 streak과 동급 보너스 → "이미 올라간 후" 추격매수 억제
3. **티어별 차등**: 대형주는 수급·이익모멘텀 중심, 소형주는 기술·밸류에이션 중심
4. **단기추천 안전장치**: 거래대금 200억 고정(유동성), 종가위치 하한 0.4(음봉전환 방지)
