추가 기능 제안

모의투자 시뮬레이터에 유용할 기능들을 골라주세요 (복수 선택 가능)

A

AI 신호 연동 알림

내 포트에 있는 종목에 AI 매도 신호가 뜨면 하이라이트 표시. 매수 전 해당 종목의 AI 신호 이력도 한눈에 확인 가능.

B

손절/익절 알림선

매수 시 손절가(-5%)와 익절가(+10%) 자동 설정. 현재가가 도달하면 포트 목록에서 경고 표시. 비율은 수동 조정 가능.

C

포트 성과 비교 차트

여러 포트의 누적 수익률을 하나의 라인 차트에 겹쳐서 비교. 어떤 전략이 더 좋은 성과인지 한눈에 파악.

D

매매 메모/일지

매수/매도 시 이유를 메모로 남기기. 나중에 복기할 때 "왜 이걸 샀지?" 확인 가능. 포트별 매매 일지 타임라인 제공.

E

포트 복제/템플릿

잘 되는 포트를 복제해서 약간 변형하며 테스트. 또는 "배당주 포트" 같은 템플릿으로 빠르게 시작.

F

벤치마크 비교

내 포트 수익률 vs 코스피/코스닥 지수 수익률을 같은 기간 기준으로 비교. 시장 대비 초과 수익 확인.
